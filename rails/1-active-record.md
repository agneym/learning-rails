# Active Record

These are the **Models**.`

- Active Record pluralizes the name of the Model to create the database table name.
- If there are two words, PascalCase them.

| Model Name | Database table |
| ---------- | -------------- |
| Article    | articles       |
| LineItem   | line_items     |
| Deer       | deers          |
| Mouse      | mice           |
| Person     | people         |
