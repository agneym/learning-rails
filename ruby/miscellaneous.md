# Miscellaneous Notes

## Puts, Print, p, pp

**Puts:**
* Adds an extra newline at the end
* Converts everything to string
* Returns `nil`

**Print**:
* Converts everything to string
* Returns `nil`

**p**:
* Does not convert to string
* Used for debugging
* Returns the object passed to it for printing

**pp**:
* Pretty Print
* Same as `p`
